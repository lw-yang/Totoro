# Totoro

Totoro is a mall system that uses SpringBoot and Vue

基于商城系统的SpringBoot+Vue前后端分离全栈开发

## Quickstart

`
git clone https://github.com/lw-yang/totoro.git`
`
cd totoro
mvn install
mvn clean package
`
